# home-manager/modules/opencode/ — OpenCode HM subtree

Configures `programs.opencode`. Wrapper + code-indexer daemon + `index_repo.py` are described in `../../../docs/routes.md`.

## Composition (`default.nix`)

- `providerDefs` = `import ./providers/<id>.nix` per provider (litellm, pollinations, aigate, aerolink),
  each returning `{ id; provider; }`; merged into `settings.provider` via `lib.listToAttrs`.
- `indexerPkg = pkgs.index-repo`, shared down via `_module.args.indexerPkg`.
- `agents.nix` is currently commented out in `default.nix`.

## Adding a provider

1. `providers/<id>.nix` returning `{ id; provider; }`.
2. Add `(import ./providers/<id>.nix)` to `providerDefs` in `default.nix`.

## Gotchas

- Model lists are generated by `../../../codegen/update-<provider>-models.sh` (pollinations/aigate/aerolink) — do not hand-edit them.
- Reads NixOS config via `osConfig`; secrets are provisioned system-side in `../../../modules/opencode-secrets.nix`.
